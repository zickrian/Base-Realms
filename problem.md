1. Standarisasi Pop-up Curious
	•	Semua pop-up Curious harus mengikuti standar pop-up yang lain.
	•	Tidak boleh ada sisi hitam di samping (jangan terlihat seperti layar baru).
	•	Tetap dianggap sebagai pop-up, bukan new screen.
	•	Konten di dalam pop-up harus bisa di-scroll.

⸻

2. Link Curious Sandbox (Fitur Copy)
	•	Link tidak boleh menggunakan emote sebagai elemen copy.
	•	Harus bisa di-copy di mobile, bukan hanya di PC.
	•	Semua emote di bagian ini dihilangkan.

⸻

3. Emote & Pop-up Sukses
	•	Emote trompet (sukses) dihapus karena mengganggu.
	•	Pop-up sukses harus mengikuti standar pop-up lain:
	•	Tidak ada sisi hitam
	•	Tetap pop-up, bukan tampilan layar baru

⸻

4. Optimasi Route NFT
	•	Route ke halaman/fitur NFT jangan lambat saat refresh.
	•	Perlu optimasi supaya load lebih cepat & responsif.

⸻

5. Standarisasi UI Battle
	•	UI pada bagian battle perlu disesuaikan dengan standar UI lain.
	•	Warna saat ini belum konsisten dengan sistem utama.
	•	Tidak menggunakan emote.
	•	Gunakan font pixel standar sesuai tema game pixel.

⸻

6. Start Battle Button (Grid Pixel)
	•	Pada pop-up Start Battle (Start & Cancel):
	•	Grid/layout tombol belum pixel style
	•	Harus disesuaikan dengan grid pixel standar UI game.

⸻

7. Tampilan Saldo IDRX
	•	Saldo IDRX ditampilkan sebagai angka bulat penuh.
	•	Contoh tampilan yang benar:
	•	100
	•	250
	•	1,000
	•	10,000
	•	Tidak ada angka desimal panjang seperti format crypto (misalnya 100.000123).
